This is a simple inventory management program.
It is written in Python 3.8

Modules required: 
    1. json
    2. os

There is a seperate 'json_maker.py' program. It can be used to make personal json files and test the 'main.py' program with different and larger data.
The order of the data in the json file is: """ id: ["name", "price", "quantity", "GST", "time of purchase"] """
